JavaScript

JavaScript is one of the most popular programming languages in the world. It is a high-level, often just-in-time compiled language that conforms to the ECMAScript standard. It has dynamic typing, prototype-based object-orientation, and first-class functions. It is used to create interactive web pages, mobile applications, in server development.

React Native

React Native is an open-source UI software framework. It is used to develop applications for Android, Android TV, iOS, macOS, tvOS, Web, by enabling developers to use the React framework along with native platform capabilities.

PHP

PHP is a general-purpose scripting programming language geared towards web development. As of March 2021, PHP was used as the server-side programming language on 79.1% of az, down from 83.5% previously, where the language could be determined, and PHP 7 is the most used version of the language with 50.3% of all websites on the web are using that version.

Node JS

Node.js is an open source server environment that allows the creation of Web servers and networking tools using JavaScript and a collection of "modules" that handle various core functionalities.

React

React is a free and open-source front-end JavaScript library for building user interfaces based on UI components. React can be used as a base in the development of single-page, mobile, or server-rendered applications with frameworks like Next.js.

Vue

Vue.js - the Progressive JavaScript Framework. An approachable, performant and versatile framework for building web user interfaces. Vue.js allows for extending HTML with HTML attributes
